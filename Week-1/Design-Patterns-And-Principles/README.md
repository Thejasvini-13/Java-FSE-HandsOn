# Design Patterns and Principles

This folder contains hands-on exercises related to Design Patterns and Principles.

## Exercises

### Exercise 1: Singleton Pattern
Objective:
Implement the Singleton Design Pattern to ensure that only one instance of the Logger class is created and used throughout the application.

### Exercise 2: Factory Method Pattern
Objective:
Implement the Factory Method Pattern to create different document types (Word, PDF, and Excel) using their respective factory classes.
